**dotfiles**

====

_public repo of my .bash_files_

### Instructions

### Quickly install with Git

```bash
git clone https://github.com/sund/dotfiles.git && cd dotfiles && source bootstrap.sh
```

To update, `cd` into your local `dotfiles` repository and then:

```bash
source bootstrap.sh
```
